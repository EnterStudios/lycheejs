## Font Generation

Custom Font needs to be installed first, it
is stored in `asset/KenVectorFuture.ttf`.

The `ui/entity/Label.fnt` is generated with:

- `family` set to `KenVector Future`
- `size` set to `32 (px)`
- `style` set to `normal`
- `spacing` set to `4 (px)`
